# Investor Tracker

A simple PyQt5 desktop app to calculate profit/loss for your VOO/QQQ investments.

## Features

- Input ticker symbol, buy price, number of shares, and current price
- Calculates total P&L (USD) and return percentage
- Green/red color indicator for profit or loss

## Requirements

- Python 3.x
- PyQt5

## Installation

```bash
pip install PyQt5
```

## Usage

```bash
python investment-tracker.py
```
